# Introduction myself

我是 LukeTseng，一名即將就讀於某國立大學資訊相關科系的高中生。

2024/02/23 開始，為了升學與精進自身的程式設計能力，我開始學著別人自己寫文章、做筆記並放到網路上。

當時我仍舊是個程式、電腦小白（雖然現在也是 XD），但我仍努力改變現狀，希望能成就更好的自己。

原本 2024 年在 HackMD 上做筆記，並在 wordpress 上架站，後來覺得 wordpress 實在限制太多了，又偶然認識到了 github page，剛好利用 Hexo 架站。

所以也就有了現在這個網站。

網站持續建置中... 持續努力跟學習程式設計、資訊工程的相關課程！

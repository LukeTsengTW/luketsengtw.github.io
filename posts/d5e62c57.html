路由器（Router）

圖片來源：https://www.ithome.com.tw/review/96747

路由器的主要功能是連接不同的網路。

詳細一點，就是一個負責將資料封包（Packets）從一個網路傳送到另一個網路的設備。

來做個比喻：

  • 網際網路（Internet）：一顆地球，是一整個世界。
  • 家中網路（LAN：Local-Area Networks 區域網路）：一棟大樓。
  • 數據機（Modem）：大樓的大門，把外面的路（電話線 / 光纖）連進來。
  • 路由器（Router）：大樓裡的收發室管理員。

當外面（網際網路）寄來一封信（資料）時，信封上只寫了大樓的地址（公開 IP）。路由器的工作就是看這封信具體是要給 101 室的 XX（手機），還是 202 室的 YY（電腦），然後精準的把信送到對應的人手上。

如果沒有路由器，信件進了大樓後就不知道該給誰，大家也無法同時寄信出去。

路由器的四大功能

連接不同的網路

這個是大家普遍都知道的功能，主要連接兩個網路：

  1. WAN（Wide Area Network）：廣域網路，外部的網際網路（連向電信業者）。
  2. LAN（Local Area Network）：區域網路，家裡或公司內部的設備連接成的網路。

路徑選擇（Routing）

這是路由器最核心的功能，也是因為這樣所以才叫做路由器 Router。

當資料要在網路上傳輸時，路由器會查看路由表（Routing Table），決定封包該走哪條路最快、最順暢。他會避開擁塞的路徑，就像 Google Maps 幫你導航一樣，只是 Google Map 可能會導航到海裡面，而 Router 不會。

網路位址轉換（NAT - Network Address Translation）

電信業者通常只給一個對外的地址（Public IP），但家裡有電腦、手機、電視等多個設備。

路由器的 NAT 功能，就是把這「唯一的一個對外地址」轉換成「多個內部地址（Private IP）」，讓所有設備可以共用一條網路線上網。

指派 IP 地址（DHCP Server）

當手機連上 Wi-Fi 時，不需要手動輸入 IP 設定，因為路由器內建的 DHCP 功能會自動分配一個臨時的內部 IP 給你的手機（如 192.168.1.5）。

路由器的種類

  1. 有線路由器（Wired Router）：只有網路孔，沒有 Wi-Fi 天線。通常用於企業，穩定性極高，專門處理大量的流量。
  2. 無線路由器（Wireless Router）：平常俗稱的 Wi-Fi 分享器，在有線路由器的基礎上增加了無線發射模組，讓手機可無線上網。
  3. 核心路由器（Core Router）：電信業者（如 Google、中華電信）機房裡用的怪獸級設備，負責處理網際網路主幹道上的超大量數據，速度極快，體積通常很大。

Router 在 TCP / IP

Router 主要運作在 TCP / IP 模型中的 Network layer。

因為 Router 主要是看 IP 地址（邏輯地址）來運作的，Router 會拆開資料封包，讀取 IP Header 中的「來源 IP」與「目的 IP」，這正是第三層 Network layer（以學術上五層 TCP / IP 模型為例）的功能。

現在常看到的一些設備，像是什麼 Wi-Fi 分享器，電信業者像中華電信的小烏龜等等，這些東西可能同時具備數據機、路由器等的功能。但我們還是習慣叫他是 Router 或分享器。

總結

四大功能

  1. 連接不同網路（WAN ↔ LAN）

路由器通常連接兩端：

  • WAN：外部網際網路（連到電信業者）
  • LAN：家中、公司內部的裝置網路

為 WAN、LAN 兩個不同世界的橋樑。

  1. 路徑選擇（Routing）

路由器查閱 Routing Table，替資料挑選最佳路徑，避開壅塞與延遲。

概念類似導航系統，但更準、更穩。

  1. 網路位址轉換（NAT）

電信業者往往只給你一組 Public IP。

NAT 讓多台裝置能共享同一個外部 IP，每台裝置在內部使用 Private IP。

  1. 指派內部 IP（DHCP Server）

裝置連上網路時，路由器自動發給它一組 IP（如 192.168.1.x），免去手動設定。

路由器的種類

  1. 有線路由器

無 Wi-Fi，穩定且可處理大量流量，多用於企業環境。

  1. 無線路由器（Wi-Fi Router）

最常見。具備路由功能並提供無線網路。

  1. 核心路由器（Core Router）

電信業者在用的，效能極強，負責處理巨量資料。

2025 VMWare Workstation Pro 免費安裝教學

大家好，我是 LukeTseng，現在的 VMWare Workstation Pro 已經轉免費了，但是安裝流程有些繁瑣，在此我來幫助各位解決這個問題！

本篇教學主要以新手向為主，透過一步、一做的流程進行教學。

若本篇教學文章有誤，懇請勘誤，感謝您。

先開啟 VT 虛擬化技術功能！

我們要玩虛擬機之前，無論是你曾玩過什麼 BlueStacks、夜神模擬器，那些都需要開啟所謂的 VT 虛擬化技術，才能順跑。

不開也是可以，只是會讓你用到懷疑人生。

我如何確認已經開啟 VT？很簡單，先到我們的工作列空白處按下滑鼠右鍵，開啟工作管理員。

開啟之後在視窗左側欄位點選【效能】選項。

點擊我們的 CPU 欄位，可看到模擬是否已啟用，啟用的話就是有開 VT。

在開啟這個之前呢，我們需要進入電腦的 BIOS 系統，以下是個人電腦（PC）與筆電端的操作方式：

  • HP：https://www.hp.com/hk-zh/shop/tech-takes/post/%E5%A6%82%E4%BD%95%E5%9C%A8-windows-%E5%80%8B%E4%BA%BA%E9%9B%BB%E8%85%A6%E4%B8%8A%E9%80%B2%E5%85%A5-bios-%E8%A8%AD%E7%BD%AE
  • ASUS：https://www.asus.com/tw/support/faq/1008829/
  • MSI：https://tw.msi.com/support/technical_details/MB_BIOS_Manual
  • 華擎（ASRock）：http://asrock.com.tw/support/faq.tw.asp?id=181

以下是開啟 VT 的手冊：

ASUS 主板配 Intel：https://www.asus.com/tw/support/faq/1043786/
ASUS 主板配 AMD：https://www.asus.com/tw/support/faq/1045141/#B

MSI 主板配 Intel：https://tw.msi.com/faq/faq-5390
MSI 主板配 AMD：https://tw.msi.com/faq/1627

首先我們開啟自己愛用的瀏覽器，輸入 VMWare Workstation。

點擊這個網站：

進去後點擊【DOWNLOAD NOW】：

此時會跳到另外一個網站，你會發現跟先前的網站頁面好像不太一樣，這是因為 VMWare 這家公司在 2022 年被博通（BoardCOM）給收購了：

那這時候我們將畫面移至右上角的倒三角形符號，然後就可以看到 Register 字樣進行註冊了：

註冊完成後，回到剛才叫你登入的地方，繼續登入：

登入完後會來到 Dash Board，我們看到左側黑色欄位的地方，有個 My Downloads：

會發現空無一物，然後有個藍色告示欄寫著 Free Software Downloads available HERE，我們點選 HERE。

進去之後，我們在搜尋欄輸入 VMWare Workstation 即可找到我們要下載的 VMWare Workstation Pro，我們直接點進去：

點進去後，這邊就看你的作業系統是哪一個，根據作業系統選擇安裝檔：

由於我的作業系統為 Windows，所以我選擇 Windows 安裝檔，這邊 Release 的部分我就選擇最新版：

點進來後，按下藍色雲朵開始下載：

若這邊不能下載，按不下去的話，檢查一下是否有方框要你勾起來，如果勾不起來的話，旁邊應該會有什麼協定之類的（有個 i 圖示），點進去看一下再回來，就可以勾選了。

下載完後在你下載的地方找到這個 .exe 安裝檔：

這個安裝檔基本上就是一直按下一步就對了，如果你沒什麼需求的話。

安裝完成後，我們可以在桌面上看到這個圖示，那就表示我們的 VMWare 已經裝好了。

VMWare 如何創建虛擬機

開啟軟體後，我們可以透過 Create a New Virtual Machine 選項創建一個新的虛擬機：

這邊使用第一個選項 Typical，之後按下 Next。

然後它會要求你 loading 光碟映像檔（.iso），在這邊以 Ubuntu 24.04.3 LTS 長期支援版作為示範。

ubuntu-24.04.3-desktop-amd64.iso 檔下載連結：https://ubuntu.com/download/desktop/thank-you?version=24.04.3&architecture=amd64&lts=true

loading 這個 .iso 檔後，我們就可以繼續下一步了。

下一步之後，會有一些個人化的設定，Full name 是在開機畫面顯示的用戶全名，而 username 則為登入驗證的帳號名。

下一步之後，可以自訂虛擬機名稱，以及他的安裝位置，這邊就由各位自行操作了～

再按下一步，來到要你設定磁碟大小的地方，這邊建議給他分配 30 GB 以上，不然日後要安裝一些軟體或安裝包會塞不太下，然後也不要給太多啦，30 ~ 40 GB 左右用就差不多了。

然後下面有兩個選項：【Store virtual disk as a single file】、【Split virtual disk into multiple files】。

如果要把他當工作環境用就選擇前者，否則後者。

下一步後 VMWare 會跟你確認硬體設定，如果有問題就 Customize Hardware，不然就直接 Finish。

完成之後，我們的虛擬機就建立好了！

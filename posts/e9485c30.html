2. Majority Element

Problem Sourceпјҡhttps://leetcode.com/problems/majority-element/description/?envType=problem-list-v2&envId=divide-and-conquer

йӣЈеәҰпјҡEasy

йЎҢзӣ®ж•ҳиҝ°пјҡ

зөҰе®ҡй•·еәҰзӮә n зҡ„йҷЈеҲ— numsпјҢеӣһеӮіеӨҡж•ёе…ғзҙ пјҲthe majority elementпјүгҖӮ

жүҖи¬ӮеӨҡж•ёе…ғзҙ е°ұжҳҜе…ғзҙ еҮәзҸҫи¶…йҒҺ [n / 2] ж¬ЎгҖӮдҪ еҸҜд»ҘеҒҮиЁӯе®ғйғҪжңғеҮәзҸҫеңЁйҷЈеҲ—дёӯгҖӮ

и§ЈйЎҢжҖқи·Ҝпјҡ

з”Ё Divide & Conquer еҒҡйҖҷйЎҢпјҢдёҖжЁЈжҳҜдәҢеҲҶжі•еҺ»еҒҡпјҢдё»иҰҒиҷ•зҗҶ [left, mid] е’Ң [mid + 1, right] йҖҷе…©ж®өеҚҖй–“гҖӮ

зөӮжӯўжўқд»¶жҳҜ if (left == right) return nums[left]; иЎЁзӨәе…ғзҙ еҖӢж•ёеү©дёҖеҖӢпјҢи©Іе…ғзҙ еҝ…зӮәеӨҡж•ёе…ғзҙ пјҢж•…еӣһеӮіжӯӨеҖјгҖӮ

йҒһиҝҙиҷ•зҗҶ lm е’Ң rm зҡ„йғЁеҲҶпјҲе·ҰеҸіеҚҖй–“пјүгҖӮ

д№ӢеҫҢеҒҡеҗҲдҪөеӢ•дҪңпјҢд№ҹзӣҙжҺҘеҒҡеҲӨж–·пјҡif (lm == rm) return lm; иЎЁзӨәе·ҰеҸіеӨҡж•ёе…ғзҙ зӣёеҗҢгҖӮ

еҰӮжһңдёҚеҗҢпјҢе°ұйңҖиҰҒеҺ»ж•ё lm и·ҹ rm е…ұеҮәзҸҫе№ҫж¬Ўпјҡ

4. Number of 1 Bits

Problem Sourceпјҡhttps://leetcode.com/problems/number-of-1-bits/description/?envType=problem-list-v2&envId=divide-and-conquer

йӣЈеәҰпјҡEasy

йЎҢзӣ®ж•ҳиҝ°пјҡ

зөҰе®ҡдёҖеҖӢжӯЈж•ҙж•ё nпјҢеҜ«дёҖеҖӢеҮҪж•ёеӣһеӮіе…¶дәҢйҖІдҪҚиЎЁзӨәжі•дёӯзҡ„иЁӯе®ҡдҪҚж•ёпјҲд№ҹзЁұзӮәжјўжҳҺж¬ҠйҮҚпјүгҖӮ

set bits жҢҮзҡ„жҳҜжңү 1 зҡ„дҪҚе…ғж•ёйҮҸпјҢеҰӮ 1011 е°ұжңү 3 еҖӢ set bitsпјҢйҖҷеҖӢд№ҹзЁұзӮә Hamming weightгҖӮ

зҜ„дҫӢзЁӢејҸзўји¬ӣи§Јпјҡ

йӣ–з„¶жңү std::popcountпјҲC++ 20пјү жҲ– __builtin_popcount()пјҲзңӢз·ЁиӯҜеҷЁж”ҜдёҚж”ҜжҸҙпјүеҸҜд»Ҙз”ЁпјҢдҪҶдёҠиҝ°е…©зЁ®ж–№жі•иӢҘеңЁжҹҗдәӣз«¶иіҪдёҠдҪҝз”ЁеҸҜиғҪдёҚеҲҮеҜҰйҡӣгҖӮ

жүҖд»ҘжҲ‘еҖ‘дҫҶжүӢеҲ»дёҖдёӢ popcountгҖӮ

еңЁзҙ°и¬ӣйҖҷеҖӢжј”з®—жі•д№ӢеүҚпјҢжҲ‘еҖ‘иҰҒе…ҲзҹҘйҒ“е№ҫеҖӢ bitmasksпјҡ

з¬¬дёҖиЎҢпјҲn = (n & 0x55555555) + ((n >> 1) & 0x55555555);пјүз”ЁеҲ° bitmasks 0x55пјҢжҢ‘еҮә n дёӯжүҖжңүеңЁеҘҮж•ёдҪҚпјҲbit 0, 2, 4, вҖҰпјүзҡ„ bitпјҢеҶҚжҠҠ n еҸіз§» 1 дҪҚеҫҢпјҲеҒ¶ж•ёдҪҚпјүпјҢд№ҹз”ЁзӣёеҗҢйҒ®зҪ©жҢ‘еҮәеҒ¶ж•ёдҪҚ bitпјҢжңҖеҫҢе…©иҖ…зӣёеҠ гҖӮ

йҖҷдёҖеұӨеҲҶжІ»дё»иҰҒе°Ү 32 еҖӢ bit еҲҶжҲҗ 16 зө„пјҢжҜҸзө„ 2bitгҖӮ

з¬¬дәҢиЎҢпјҲn = (n & 0x33333333) + ((n >> 2) & 0x33333333);пјүз”ЁеҲ° bitmasks 0x33пјҢе°ҮеүӣжүҚз®—еҘҪзҡ„жҜҸ 2bit зө„еҒҡзҙҜеҠ ж“ҚдҪңпјҢжҠҠзӣёй„°зҡ„е…©еҖӢ 2bit зө„еҗҲжҲҗдёҖеҖӢ 4bit зө„дёҰзҙҜеҠ  1 зҡ„ж•ёйҮҸгҖӮ

з„¶еҫҢйҖҷеұӨеҲҶжІ»д№ҹе°Ү 32bit еҲҶжҲҗ 8 зө„пјҢжҜҸзө„ 4bitпјҢиЁҲз®—жҜҸзө„е…§ 1 зҡ„еҖӢж•ёгҖӮ

з¬¬дёүиЎҢпјҲn = (n & 0x0f0f0f0f) + ((n >> 4) & 0x0f0f0f0f);пјүз”ЁеҲ° bitmasks 0x0fпјҢе°Үзӣёй„°зҡ„е…©еҖӢ 4bit зө„еҗҲжҲҗдёҖеҖӢ 8bit зө„пјҢеҗҢжЁЈдҪҝз”Ё bitmasks еҲҶдҪҺ 4bit еҸҠй«ҳ 4bit зӣёеҠ гҖӮ

и©ІеұӨеҲҶжІ»еҶҚжҠҠ 32bit еҲҶжҲҗ 4 зө„пјҢжҜҸзө„ 8bitпјҢиЁҲз®—жҜҸзө„е…§ 1 зҡ„еҖӢж•ёгҖӮ

з¬¬еӣӣиЎҢпјҲn = (n & 0x00ff00ff) + ((n >> 8) & 0x00ff00ff);пјүд№ҹз”ЁеҲ°дёҖжЁЈзҡ„ bitmasksпјҢеҸӘдёҚйҒҺеӨҡеҠ еҖӢ fпјҢеңЁдәҢйҖІдҪҚдёҠе°ұеӨҡдәҶеӣӣеҖӢ fгҖӮ

йҖҷеұӨеҲҶжІ»жҠҠе°Үзӣёй„°е…©еҖӢ 8bit зө„еҗҲжҲҗ 1 еҖӢ 16bit зө„пјҢеҶҚзҙҜеҠ еҗ„ 8bit дёӯ 1 зҡ„еҖӢж•ёгҖӮ

з¬¬дә”иЎҢпјҲn = (n & 0x0000ffff) + ((n >> 16) & 0x0000ffff);пјүз”ЁеҲ°жңҖеҫҢдёҖеҖӢ bitmasks 0xffпјҢе°Үе…©еҖӢ 16bit зө„еҗҲжҲҗж•ҙеҖӢ 32bit зҡ„зө„еҗҲпјҢеҶҚе°ҮеүҚеҫҢ 16bit дёӯ 1 зҡ„еҖӢж•ёеҗҲдҪөгҖӮ

жңҖзөӮжҲ‘еҖ‘е°ұеҫһ 32 bit иЈЎйқўеҫ—еҲ°жүҖжңү 1 зҡ„ж•ёйҮҸдәҶпјҢзӮәд»ҖйәјжҳҜ 32 bit е‘ўпјҹеӣ зӮәйЎҢзӣ®иҰҒжұӮ intпјҢint е°ұеүӣеҘҪ 32 bitгҖӮ

е®Ңж•ҙзҜ„дҫӢзЁӢејҸзўјпјҡ

5. Longest Nice Substring

Problem Sourceпјҡhttps://leetcode.com/problems/longest-nice-substring/description/?envType=problem-list-v2&envId=divide-and-conquer

йӣЈеәҰпјҡEasy

йЎҢзӣ®ж•ҳиҝ°пјҡ

дёҖеӯ—дёІ s еҰӮжһңжҳҜ niceпјҢйӮЈе°Қж–јжҜҸеҖӢеӯ—жҜҚиЎЁдёӯзҡ„жҜҸеҖӢеӯ—жҜҚйғҪеҢ…еҗ« sпјҢе®ғд»ҘеӨ§еҜ«е’Ңе°ҸеҜ«еҪўејҸеҮәзҸҫгҖӮ

иҲүдҫӢдҫҶиӘӘпјҢ"abABB" жҳҜ niceпјҢеӣ зӮә 'A' е’Ң 'a' еҮәзҸҫпјҢдёҰдё” 'B' е’Ң 'b' еҮәзҸҫгҖӮз„¶иҖҢпјҢ"abA" дёҚжҳҜ niceпјҢеӣ зӮә 'b' еҮәзҸҫпјҢиҖҢ 'B' жІ’жңүеҮәзҸҫгҖӮ

зөҰе®ҡдёҖеҖӢеӯ—дёІ sпјҢеӣһеӮі s дёӯжңҖй•·зҡ„гҖҒж»ҝи¶і nice жўқд»¶зҡ„еӯҗеӯ—дёІгҖӮеҰӮжһңжңүеӨҡеҖӢпјҢеүҮеӣһеӮіжңҖж—©еҮәзҸҫзҡ„еӯҗеӯ—дёІгҖӮеҰӮжһңжІ’жңүпјҢеүҮеӮіеӣһз©әеӯ—дёІгҖӮ

и§ЈйЎҢжҖқи·Ҝпјҡ

жҲ‘еҖ‘з”ЁиіҮж–ҷзөҗж§Ӣ unordered_set <int> seen(s.begin(), s.end()); дҫҶеҜҰдҪңйҖҷйЎҢгҖӮ

йҰ–е…Ҳи·‘иҝҙеңҲпјҢе…§йғЁеҲӨж–·дёҖеҖӢеӯ—дёІдёӯзҡ„еӯ—е…ғ c жңүжІ’жңүе…¶д»–зӣёеҸҚзҡ„еӨ§е°ҸеҜ«пјҢеҰӮ A иҰҒжүҫеҲ°е°ҚжҮүзҡ„е°ҸеҜ« a жүҚжҳҜдёҖеҖӢ nice substringгҖӮпјҲйҖҷйӮҠиҰҒжүҫд»–е°ҚжҮүзӣёеҸҚзҡ„еӨ§е°ҸеҜ«пјҢжҲ‘еҖ‘з”Ё seen.find() еҺ»жүҫпјү

еӨ§е°ҸеҜ«еҲӨж–·еҸҜд»ҘйЎҚеӨ–еҜ«дёҖеҖӢ toggleCase() еҮҪж•ёеҺ»еҲӨж–·пјҢеҰӮпјҡ

еҰӮжһң seen.find() жІ’жңүжүҫеҲ°пјҢеүҮеӣһеӮі end() йҖҷеҖӢ iteratorгҖӮ

жІ’жүҫеҲ°зҡ„и©ұжҲ‘еҖ‘е°ұиҰҒеҒҡйҒһиҝҙеҺ»жүҫдәҶпјҢйҖҷйӮҠз”ЁеҲҶжІ»жі•зҡ„жҰӮеҝөе°Үеӯ—дёІеҲҮе…©еҚҠеҺ»жүҫпјҡ

substr(0, i) зӮәеҫһзҙўеј• 0 й–Ӣе§ӢпјҢж“·еҸ–й•·еәҰ i зҡ„еӯҗеӯ—дёІпјҲдёҚеҗ«зҙўеј• i зҡ„еӯ—е…ғпјүпјҢеҪўжҲҗе·ҰеҚҠйӮҠзҡ„еӯҗеӯ—дёІгҖӮ

substr(i + 1) еҫһзҙўеј• i + 1 й–Ӣе§ӢпјҢж“·еҸ–еҲ°еӯ—дёІе°ҫзҡ„еӯҗеӯ—дёІпјҢеҪўжҲҗеҸіеҚҠйӮҠеӯҗеӯ—дёІгҖӮ

жңҖеҫҢеӣһеӮіијғй•·зҡ„еӯҗеӯ—дёІпјҡreturn left.size() >= right.size() ? left : rightгҖӮ

жңҖеҫҢзҡ„жңҖеҫҢпјҢеҰӮжһңжҲ‘еҖ‘еүҚйқўеҲӨж–·йғҪжІ’жңүеӣһеӮі end() зҡ„и©ұпјҢйӮЈе°ұиЎЁзӨәиӘӘж•ҙдёІеӯ—дёІе®ҢзҫҺз¬ҰеҗҲ nice substring зҡ„жўқд»¶пјҢе°ұзӣҙжҺҘ return sгҖӮ

ТЅђУгѓтцџТЋИтЁЃу┤ат░▒Тў»тЁЃу┤атЄ║уЈЙУХЁжЂј [n / 2] ТгАсђѓСйатЈ»С╗ЦтЂЄУеГт«ЃжЃйТюЃтЄ║уЈЙтюежЎБтѕЌСИГсђѓ

ухѓТГбТбЮС╗ХТў» if (left == right) return nums[left]; УАеуц║тЁЃу┤атђІТЋИтЅЕСИђтђІ№╝їУЕ▓тЁЃу┤ат┐Ёуѓ║тцџТЋИтЁЃу┤а№╝їТЋЁтЏътѓ│ТГцтђ╝сђѓ

4. Number of 1 Bits

Problem Source№╝џhttps://leetcode.com/problems/number-of-1-bits/description/?envType=problem-list-v2&envId=divide-and-conquer

жЏБт║д№╝џEasy

жАїуЏ«ТЋўУ┐░№╝џ

ухдт«џСИђтђІТГБТЋ┤ТЋИ n№╝їт»ФСИђтђІтЄйТЋИтЏътѓ│тЁХС║їжђ▓СйЇУАеуц║Т│ЋСИГуџёУеГт«џСйЇТЋИ№╝ѕС╣Ъуе▒уѓ║Т╝бТўјТгіжЄЇ№╝Ѕсђѓ

set bits ТїЄуџёТў»ТюЅ 1 уџёСйЇтЁЃТЋИжЄЈ№╝їтдѓ 1011 т░▒ТюЅ 3 тђІ set bits№╝їжђЎтђІС╣Ъуе▒уѓ║ Hamming weightсђѓ

у»ёСЙІуеІт╝Јуб╝УгЏУДБ№╝џ

жЏќуёХТюЅ std::popcount№╝ѕC++ 20№╝Ѕ Тѕќ __builtin_popcount()№╝ѕуюІуиеУГ»тЎеТћ»СИЇТћ»ТЈ┤№╝ЅтЈ»С╗Цуће№╝їСйєСИіУ┐░тЁЕуе«Тќ╣Т│ЋУІЦтюеТЪљС║ЏуФХУ│йСИіСй┐ућетЈ»УЃйСИЇтѕЄт»джџЏсђѓ

ТЅђС╗ЦТѕЉтђЉСЙєТЅІтѕ╗СИђСИІ popcountсђѓ

тюеу┤░УгЏжђЎтђІТ╝ћу«ЌТ│ЋС╣ІтЅЇ№╝їТѕЉтђЉУдЂтЁѕуЪЦжЂЊт╣ЙтђІ bitmasks№╝џ

уггСИђУАї№╝ѕn = (n & 0x55555555) + ((n >> 1) & 0x55555555);№╝Ѕућетѕ░ bitmasks 0x55№╝їТїЉтЄ║ n СИГТЅђТюЅтюетЦЄТЋИСйЇ№╝ѕbit 0, 2, 4, Рђд№╝Ѕуџё bit№╝їтєЇТіі n тЈ│уД╗ 1 СйЇтЙї№╝ѕтЂХТЋИСйЇ№╝Ѕ№╝їС╣ЪућеуЏИтљїжЂ«уйЕТїЉтЄ║тЂХТЋИСйЇ bit№╝їТюђтЙїтЁЕУђЁуЏИтіасђѓ

жђЎСИђт▒цтѕєТ▓╗СИ╗УдЂт░Є 32 тђІ bit тѕєТѕљ 16 ухё№╝їТ»Јухё 2bitсђѓ

уггС║їУАї№╝ѕn = (n & 0x33333333) + ((n >> 2) & 0x33333333);№╝Ѕућетѕ░ bitmasks 0x33№╝їт░ЄтЅЏТЅЇу«ЌтЦйуџёТ»Ј 2bit ухётЂџу┤»тіаТЊЇСйю№╝їТііуЏИжё░уџётЁЕтђІ 2bit ухётљѕТѕљСИђтђІ 4bit ухёСИду┤»тіа 1 уџёТЋИжЄЈсђѓ

уёХтЙїжђЎт▒цтѕєТ▓╗С╣Ът░Є 32bit тѕєТѕљ 8 ухё№╝їТ»Јухё 4bit№╝їУеѕу«ЌТ»ЈухётЁД 1 уџётђІТЋИсђѓ

уггСИЅУАї№╝ѕn = (n & 0x0f0f0f0f) + ((n >> 4) & 0x0f0f0f0f);№╝Ѕућетѕ░ bitmasks 0x0f№╝їт░ЄуЏИжё░уџётЁЕтђІ 4bit ухётљѕТѕљСИђтђІ 8bit ухё№╝їтљїТеБСй┐уће bitmasks тѕєСйј 4bit тЈіжФў 4bit уЏИтіасђѓ

УЕ▓т▒цтѕєТ▓╗тєЇТіі 32bit тѕєТѕљ 4 ухё№╝їТ»Јухё 8bit№╝їУеѕу«ЌТ»ЈухётЁД 1 уџётђІТЋИсђѓ

уггтЏЏУАї№╝ѕn = (n & 0x00ff00ff) + ((n >> 8) & 0x00ff00ff);№╝ЅС╣Ъућетѕ░СИђТеБуџё bitmasks№╝їтЈфСИЇжЂјтцџтіатђІ f№╝їтюеС║їжђ▓СйЇСИіт░▒тцџС║єтЏЏтђІ fсђѓ

жђЎт▒цтѕєТ▓╗Тііт░ЄуЏИжё░тЁЕтђІ 8bit ухётљѕТѕљ 1 тђІ 16bit ухё№╝їтєЇу┤»тіатљё 8bit СИГ 1 уџётђІТЋИсђѓ

уггС║ћУАї№╝ѕn = (n & 0x0000ffff) + ((n >> 16) & 0x0000ffff);№╝Ѕућетѕ░ТюђтЙїСИђтђІ bitmasks 0xff№╝їт░ЄтЁЕтђІ 16bit ухётљѕТѕљТЋ┤тђІ 32bit уџёухётљѕ№╝їтєЇт░ЄтЅЇтЙї 16bit СИГ 1 уџётђІТЋИтљѕСйхсђѓ

ТюђухѓТѕЉтђЉт░▒тЙъ 32 bit УБАжЮбтЙЌтѕ░ТЅђТюЅ 1 уџёТЋИжЄЈС║є№╝їуѓ║С╗ђж║╝Тў» 32 bit тЉб№╝ЪтЏауѓ║жАїуЏ«УдЂТ▒ѓ int№╝їint т░▒тЅЏтЦй 32 bitсђѓ

т«їТЋ┤у»ёСЙІуеІт╝Јуб╝№╝џ

5. Longest Nice Substring

Problem Source№╝џhttps://leetcode.com/problems/longest-nice-substring/description/?envType=problem-list-v2&envId=divide-and-conquer

жЏБт║д№╝џEasy

жАїуЏ«ТЋўУ┐░№╝џ

СИђтГЌСИ▓ s тдѓТъюТў» nice№╝їжѓБт░ЇТќ╝Т»ЈтђІтГЌТ»ЇУАеСИГуџёТ»ЈтђІтГЌТ»ЇжЃйтїЁтљФ s№╝їт«ЃС╗ЦтцДт»Фтњїт░Јт»Фтйбт╝ЈтЄ║уЈЙсђѓ

УѕЅСЙІСЙєУфф№╝ї"abABB" Тў» nice№╝їтЏауѓ║ 'A' тњї 'a' тЄ║уЈЙ№╝їСИдСИћ 'B' тњї 'b' тЄ║уЈЙсђѓуёХУђї№╝ї"abA" СИЇТў» nice№╝їтЏауѓ║ 'b' тЄ║уЈЙ№╝їУђї 'B' Т▓њТюЅтЄ║уЈЙсђѓ

ухдт«џСИђтђІтГЌСИ▓ s№╝їтЏътѓ│ s СИГТюђжЋиуџёсђЂТ╗┐УХ│ nice ТбЮС╗ХуџётГљтГЌСИ▓сђѓтдѓТъюТюЅтцџтђІ№╝їтЅЄтЏътѓ│ТюђТЌЕтЄ║уЈЙуџётГљтГЌСИ▓сђѓтдѓТъюТ▓њТюЅ№╝їтЅЄтѓ│тЏъуЕ║тГЌСИ▓сђѓ

УДБжАїТђЮУи»№╝џ

ТѕЉтђЉућеУ│ЄТќЎухљТДІ unordered_set <int> seen(s.begin(), s.end()); СЙєт»дСйюжђЎжАїсђѓ

ждќтЁѕУиЉУ┐┤тюѕ№╝їтЁДжЃетѕцТќиСИђтђІтГЌСИ▓СИГуџётГЌтЁЃ c ТюЅТ▓њТюЅтЁХС╗ќуЏИтЈЇуџётцДт░Јт»Ф№╝їтдѓ A УдЂТЅЙтѕ░т░ЇТЄЅуџёт░Јт»Ф a ТЅЇТў»СИђтђІ nice substringсђѓ№╝ѕжђЎжѓіУдЂТЅЙС╗ќт░ЇТЄЅуЏИтЈЇуџётцДт░Јт»Ф№╝їТѕЉтђЉуће seen.find() тј╗ТЅЙ№╝Ѕ

тцДт░Јт»ФтѕцТќитЈ»С╗ЦжАЇтцќт»ФСИђтђІ toggleCase() тЄйТЋИтј╗тѕцТќи№╝їтдѓ№╝џ

тдѓТъю seen.find() Т▓њТюЅТЅЙтѕ░№╝їтЅЄтЏътѓ│ end() жђЎтђІ iteratorсђѓ

Т▓њТЅЙтѕ░уџёУЕ▒ТѕЉтђЉт░▒УдЂтЂџжЂъУ┐┤тј╗ТЅЙС║є№╝їжђЎжѓіућетѕєТ▓╗Т│ЋуџёТдѓт┐хт░ЄтГЌСИ▓тѕЄтЁЕтЇітј╗ТЅЙ№╝џ

substr(0, i) уѓ║тЙъу┤бт╝Ћ 0 жќІтДІ№╝їТЊитЈќжЋит║д i уџётГљтГЌСИ▓№╝ѕСИЇтљФу┤бт╝Ћ i уџётГЌтЁЃ№╝Ѕ№╝їтйбТѕљтидтЇіжѓіуџётГљтГЌСИ▓сђѓ

substr(i + 1) тЙъу┤бт╝Ћ i + 1 жќІтДІ№╝їТЊитЈќтѕ░тГЌСИ▓т░ЙуџётГљтГЌСИ▓№╝їтйбТѕљтЈ│тЇіжѓітГљтГЌСИ▓сђѓ

ТюђтЙїтЏътѓ│У╝ЃжЋиуџётГљтГЌСИ▓№╝џreturn left.size() >= right.size() ? left : rightсђѓ

ТюђтЙїуџёТюђтЙї№╝їтдѓТъюТѕЉтђЉтЅЇжЮбтѕцТќижЃйТ▓њТюЅтЏътѓ│ end() уџёУЕ▒№╝їжѓБт░▒УАеуц║УффТЋ┤СИ▓тГЌСИ▓т«їуЙјугдтљѕ nice substring уџёТбЮС╗Х№╝їт░▒уЏ┤ТјЦ return sсђѓ

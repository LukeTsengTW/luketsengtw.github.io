布林代數式（Boolean Algebra）

Boolean Algebra 是數學當中的一個分支，專門用來處理 0 跟 1 這種真（True）假（False）值。主要針對二進位變數以及邏輯運算，如最基礎的 AND、OR、NOT。

邏輯運算（Logical Operations）

  • AND or Conjunction（關聯）
  • OR or Disjunction（無關）
  • NOT or Negation（否定）

AND 用布林代數式可表示為 A⋅BA \cdot BA⋅B 或是 A∧BA \wedge BA∧B 。

OR 用布林代數式可表示為 A+BA + BA+B 或是 A∨BA \lor BA∨B 。

NOT 用布林代數式可表示為 ⇁A\rightharpoondown A⇁A 或 NOTANOTANOTA ，那本系的教授是用一個 bar 來表示 NOT，如 A‾\overline{A}A 。

AND 要全部輸入都是 1，輸出才會是 1。

OR 只要其中一個輸入是 1，輸出就會都是 1，要產生 0 的情況，一定要全部輸入都是 0。

NOT 會將狀態反轉，就是 0 變成 1，1 變成 0。

真值表（Truth Table）

真值表（Truth Table）是一種用於邏輯運算的數學表格，主要用來列出所有可能的輸入組合下，布林代數式的結果。

如果有兩個輸入端如 A、B，那麼其輸出的結果會是 22=42^2 = 422=4 種可能，三個輸入端就是 23=82^3 = 823=8 種可能結果，由此可以推算出他的公式應該是 2n2^n2n ，也就是輸出端的可能結果， n 是輸入端的個數。

布林代數式定律（Theorems / Laws）

  1. 同一律（Identity Law）：
    • A+0=AA + 0 = AA+0=A
    • A⋅1=AA \cdot 1 = AA⋅1=A
  2. 交換律（Commutative Law）：
    • A⋅B=B⋅AA \cdot B = B \cdot AA⋅B=B⋅A
    • A+B=B+AA + B = B + AA+B=B+A
  3. 結合律（Associative Law）：
    • (A⋅B)⋅C=A⋅(B⋅C)(A \cdot B) \cdot C = A \cdot (B \cdot C)(A⋅B)⋅C=A⋅(B⋅C)
    • (A+B)+C=A+(B+C)(A + B) + C = A + (B + C)(A+B)+C=A+(B+C)
  4. 分配律（Distributive Law）：
    • A⋅(B+C)=(A⋅B)+(A⋅C)A \cdot (B + C) = (A \cdot B) + (A \cdot C)A⋅(B+C)=(A⋅B)+(A⋅C)
  5. 反轉律（Inversion Law）：
    • 註：經過兩次 NOT 還是自己。
    • A‾‾=A\overline{\overline{A}} = AA=A
  6. 補數律（Complement Law）：
    • A+A‾=1A + \overline{A} = 1A+A=1
    • A⋅A‾=0A \cdot \overline{A} = 0A⋅A=0
  7. 冪等律（Idempotent Law）：
    • A+A=AA + A = AA+A=A
    • A⋅A=AA \cdot A = AA⋅A=A
  8. 笛摩根定理（De Morgan’s Laws）：
    • A⋅B‾=A‾+B‾\overline{A \cdot B} = \overline{A} + \overline{B}A⋅B=A+B
    • A+B‾=A‾⋅B‾\overline{A + B} = \overline{A} \cdot \overline{B}A+B​=A⋅B

學這要幹嘛？可以簡化布林代數式，也更進一步去改善、精簡電路設計，降低開發成本。

補充說明：XOR 定律，基本上與布林代數式定律差不多，但些許地方有差別。

  1. 自反律（Self-Inverse Law）：
    • A⊕A=0A \oplus A = 0A⊕A=0
  2. 交換律（Commutative Law）：
    • A⊕B=B⊕AA \oplus B = B \oplus AA⊕B=B⊕A
  3. 結合律（Associative Law）：
    • A⊕B⊕C=A⊕(B⊕C)=(A⊕B)⊕CA \oplus B \oplus C = A \oplus (B \oplus C) = (A \oplus B) \oplus CA⊕B⊕C=A⊕(B⊕C)=(A⊕B)⊕C
  4. 同一律（Identity Law）：
    • A⊕0=AA \oplus 0 = AA⊕0=A

註：XOR 在密碼學領域有相當的用處，透過這些定律可以進行加密解密的動作。

總結

一、布林代數（Boolean Algebra）

布林代數是數學的分支，用於處理真（1）與假（0）的邏輯運算。

三種基本運算：

  1. AND（ ⋅\cdot⋅ ）：所有輸入為 1，輸出才為 1。
  2. OR（ +++ ）：只要有一個輸入為 1，輸出為 1。
  3. NOT（ ‾\overline{} ）：將輸入反轉，0→1、1→0。

所有的邏輯閘都可以用 AND、OR、NOT 這些邏輯閘所組成。

二、真值表（Truth Table）

真值表列出所有輸入組合的輸出結果。

若輸入數為 nnn ，則有 2n2^n2n 種輸出組合。

以下是 AND、OR、NOT 個別的真值表：

AND（ A⋅BA \cdot BA⋅B ）：

三、邏輯電路與布林代數的對應

邏輯電路可直接轉換成布林代數式，反之亦然：

  • AND→Y=A⋅BAND \rightarrow Y = A \cdot BAND→Y=A⋅B
  • OR→Y=A+BOR \rightarrow Y = A + BOR→Y=A+B
  • NOT→Y=A‾NOT \rightarrow Y = \overline{A}NOT→Y=A

四、進階邏輯運算

運算符號定義
XOR⊕兩輸入不同為真
NAND↓AND 輸出後面加上 NOT
NOR↑OR 輸出後面加上 NOT
XNOR↔兩輸入相同為真

五、布林代數基本定律（Laws of Boolean Algebra）

  1. 同一律（Identity Law）：
    • A+0=AA + 0 = AA+0=A
    • A⋅1=AA \cdot 1 = AA⋅1=A
  2. 交換律（Commutative Law）：
    • A⋅B=B⋅AA \cdot B = B \cdot AA⋅B=B⋅A
    • A+B=B+AA + B = B + AA+B=B+A
  3. 結合律（Associative Law）：
    • (A⋅B)⋅C=A⋅(B⋅C)(A \cdot B) \cdot C = A \cdot (B \cdot C)(A⋅B)⋅C=A⋅(B⋅C)
    • (A+B)+C=A+(B+C)(A + B) + C = A + (B + C)(A+B)+C=A+(B+C)
  4. 分配律（Distributive Law）：
    • A⋅(B+C)=(A⋅B)+(A⋅C)A \cdot (B + C) = (A \cdot B) + (A \cdot C)A⋅(B+C)=(A⋅B)+(A⋅C)
  5. 反轉律（Inversion Law）：
    • 註：經過兩次 NOT 還是自己。
    • A‾‾=A\overline{\overline{A}} = AA=A
  6. 補數律（Complement Law）：
    • A+A‾=1A + \overline{A} = 1A+A=1
    • A⋅A‾=0A \cdot \overline{A} = 0A⋅A=0
  7. 冪等律（Idempotent Law）：
    • A+A=AA + A = AA+A=A
    • A⋅A=AA \cdot A = AA⋅A=A
  8. 笛摩根定理（De Morgan’s Laws）：
    • A⋅B‾=A‾+B‾\overline{A \cdot B} = \overline{A} + \overline{B}A⋅B=A+B
    • A+B‾=A‾⋅B‾\overline{A + B} = \overline{A} \cdot \overline{B}A+B​=A⋅B

六、補充 XOR 定律（XOR Laws）

  1. 自反律（Self-Inverse Law）：
    • A⊕A=0A \oplus A = 0A⊕A=0
  2. 交換律（Commutative Law）：
    • A⊕B=B⊕AA \oplus B = B \oplus AA⊕B=B⊕A
  3. 結合律（Associative Law）：
    • A⊕B⊕C=A⊕(B⊕C)=(A⊕B)⊕CA \oplus B \oplus C = A \oplus (B \oplus C) = (A \oplus B) \oplus CA⊕B⊕C=A⊕(B⊕C)=(A⊕B)⊕C
  4. 同一律（Identity Law）：
    • A⊕0=AA \oplus 0 = AA⊕0=A

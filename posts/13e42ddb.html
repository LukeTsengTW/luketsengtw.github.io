【Windows 11 秘笈】重灌或裝新系統必做的五件事！

哈囉大家好，我是 LukeTseng，近期換新電腦裝系統時，發現忘記 win 11 有些操作令我不太習慣，且又忘記該如何操作，實在令人煩惱，所以便製作了這篇文章。

:::info
註：這篇文章純粹是以我個人習慣而製作的，若有些設定你覺得不習慣可改回去
:::

廢話不多說，馬上開始吧！

一、工作列置左

首先工作列靠左是從 xp 時代至 win10 時代當中漸漸習慣的一件事，身為 windows 的長期使用者，當然務必首要的就是將 win 11 這惱人的工作列置中改成靠左啦！

那其實要靠左也很簡單，首先我們透過快捷鍵 windows 鍵 + i 鍵 打開設定，或是透過工作列底下的 windows 開始鍵打開也可以。

進入 [個人化] 後，可以看到有個選項叫做 [工作列]。

點進去 [工作列] 後，可看到最底下有個叫做 [工作列行為] 的選單。

點擊一下後，將 [工作列對齊] 中最右邊的下拉選單點擊一下，將 [置中] 改成 [靠左]。

二、右鍵選單

要改回 w10 以前時代的右鍵選單，有兩種方法可使用：

  1. 登錄編輯程式（Regedit）
  2. CMD 打指令

個人認為第一種方法對大眾而言過於複雜，所以我們就直接打指令最快。

首先透過 windows 鍵 + R 開啟 [執行]。

透過打指令的方式，如同第一種方式一般，只是透過系統自動幫你修改其中的數值。

三、五次 Shift 鍵

相信各位在打遊戲時非常討厭一件事，那就是在狂按 Shift 時，突然跳到桌面，系統跟你說你按超過五次 Shift 鍵了！再跳回去就發現你已經掛了（作者當初玩法環時遇到這個問題，非常的生氣，然後就想出這次教學XD）

我們同樣透過 windows 鍵 + i 鍵 打開設定，或是從開始鍵打開也可以。

點進去後，我們把第一個選項關閉，這樣子就不會出現小框框囉。

四、移除 Windows 更新檔

Windows 更新檔是非常龐大的垃圾，建議可以清除，那麼因為我很早就把他刪了，所以沒有圖片展示。

具體步驟如下：

打開 [系統設定] -> 點選左側攔的 [系統] -> 進入 [儲存體] -> 點選 [清理建議] -> 可看有什麼東西要刪的，沒用到可刪一刪，系統更新檔會在 [大型或未使用的檔案] 裡面。

五、更新驅動程式

時代愈加進步了，現在的 Windows 通常都會自動幫你裝上驅動程式，但有些還是要自己手動裝比較保險。

像我是 NVIDIA GPU 的忠誠使用者，每次重灌時都一定會去他的官網上下載 driver 下來：https://www.nvidia.com/zh-tw/drivers/

若有遊戲需求的話，直接裝 [GeForce Game Ready] 即可；若是工作室需求（如使用 PS、PR、繪圖軟體、渲染軟體如：Blender 3D 等），就使用 [NVIDIA Studio driver]。

A 卡有相對應的驅動可自行去安裝。

不過作者在此想講一件事，屬於自身的經驗談。因為換上 AMD CPU 的緣故，之前在玩地平線 4（Forza Horizon 4）時發生 Memory Overflow 的問題。

後來發現其實是 AMD Driver 搞的鬼，接下來請看下一節。

個人經驗談：因驅動程式而導致特定遊戲打不開

後來我透過巴哈找到一個 Solution，發現這個問題竟然就神奇的解決了！？於是在此分享給廣大使用 AMD 的網友們參考。

我不知道其他遊戲是否也會這樣，但就以我玩 FH4 的經驗來說，這問題是頭次遇到。

那接下來讓我們進入正題：

進入網站：https://www.guru3d.com/download/display-driver-uninstaller-download/

右下角有兩個鏡像的下載連結，選哪個都沒差。

下載完後可看到此壓縮檔，直接將它解壓縮。

直接執行裡面的 [DDU v18.0.9.0.exe] 7z 檔案，再次解壓縮。

直接打開資料夾 [DDU v18.0.9.0] 裡面的 [Display Driver Uninstaller.exe]，需要使用系統管理員權限開啟。

將會跳出以下畫面，直接按確定就好：

以下畫面保留預設值就好，直接按下 Close。

接下來會出現此畫面，說你不在安全模式，按下確定即可

然後在右邊選擇 [GPU]，下面記得選擇 [AMD]。

選擇完後，按下最左邊第一個按鈕 [Clean and restart]。

等它處理完，重開機後再開遊戲就沒問題囉。

# PixAI 一個免費的二次元 AI 生圖網站，使用教學與模型推薦

哈囉大家好我是 LukeTseng，今天來介紹一個免費的生圖網站叫 PixAI，他的命名神似知名的插畫交流網站 Pixiv（眾多老濕機們的平台，讚讚 der），也當然，他在二次元生圖的部分十分優秀，接下來就讓我帶各位來看這個網站。

# 為什麼要用 PixAI？

  1. AI 生圖雲端運算，不用本地端執行生圖程式
  2. 免費
  3. 功能齊全且細項多雜
  4. 能產二次元香圖，讚讚

# PixAI 生成圖片

若要生成圖片，我們可至左側欄的生成，然後點下去。

來到生成頁面後，可以看到有相當多的功能可選擇：

首先看到模型部分，點展示更多可看到更多模型：

如果往下滑會看到沒有加上 XL 的模型，在這邊的 XL 模型應該是對應到 stable diffusion 的 XL 模型，參數量高達 35 億。

沒有加 XL 的在生圖方面會比較弱一些。

而至於在每個模型的右側邊欄，有寫一些建議參數：

以及生成的圖是否可商用：

那在本篇就使用 Haruka v2 作為範例。

一樣在模型處的右側邊欄，可以選擇尺寸，若要自訂尺寸就要購買他們家的會員，也可選擇一次要生成多少張圖片：

點開高級設定後，有些預設值模型已經幫你調好了，就可以不用動。

而 Seed 的部分則是將生成的圖片固定成一種風格，VAE 模型簡單來說就像是將生成的圖片套上濾鏡。

筆者在此就不動高級設定，直接來看輸入提示詞的部分：

在提示詞的部分，大致上與 stable diffusion 操作差不多，都是輸入短語、單字為主，且每個短語與單字間要用半形逗號，作為間隔。

而提示詞的權重可使用小括號 () 和中括號 [] 做調整。小括號是增加權重，中括號為減少權重。

可以使用多個括號括起來，有疊加的效果。

而每個權重調整都是乘上 1.1 倍，像 ((a)) 的話就是增加權重 1.21 倍，為 1.1 的平方。

也可以直接指定權重值，如： (a : 1.5) ，這樣就會乘上 1.5 倍的權重。

另外提示詞的順序也有差異，擺在前面的提示詞表示為主角，是優先生成的順序，越後面則是背景、層次、細節的調整。

筆者在這邊用了以下提示詞，供給各位參考：

girl, sailor suit, yokozuwari, hand on headphones, long hair, blonde hair, castle

生圖時，有個高優先順序，其實就是讓系統把你排到優先佇列中，比較快生圖，如果想省積分的話就把他關掉（但是會久到靠邀，建議還是開一下）：

生成完後，大概會長得像這樣：

AI 圖通常有許多作畫失誤的細節，以及透視問題，那麼如何改善呢？

  1. 優化提示詞
  2. 透過 Lora 微調模型改善

# 優化提示詞

可以用 PromptPerfect 幫你做優化提示詞的動作。

具體使用方式可看我上一篇的教學：https://hackmd.io/@LukeTseng/ryfH6qHWxg

我使用 PromptPerfect 優化過的提示詞進行生圖後，如下：

Blonde girl+, sailor suit, yokozuwari pose, hand on headphones, long hair, floating castle in background, serene atmosphere, (high detail)+, photorealistic, vivid colors

會發現完全偏離了二次元的方向耶！？原因是 PromptPerfect 加上了 photorealistic 的提示詞，才會導致這樣。

如果我們把它移除掉看會變怎樣：

優化過後在光影效果上好了很多，但是手指有時候會缺一根阿，囧 |||。

那麼就有請我們的 Lora 微調模型老大哥出來了！！

# 推薦使用的第一個 Lora 模型

這個模型叫做 ModelBoosterXL，是用於提升生圖細節的，非常好用！

# 推薦使用的第二個 Lora 模型

第二個推薦的模型叫做 Background Enhancer [SDXL]，在生成一些好看的背景圖時很好用，用下去的背景超美！

# 推薦使用的第三個 Lora 模型

第三個推薦的模型叫做 Perfect Hands SDXL，可以生成出較為優良的手部，不會缺東缺西的 XD。

# 推薦使用的第四個 Lora 模型

第四個推薦的模型叫做 XL Outline / 輪廓線，有些動漫的畫風會稍微有輪廓線存在，可根據喜好去進行添增。

# 推薦使用的第五個 Lora 模型

第五個推薦的模型叫做 Jeweled eyes / 吸い込まれそうな瞳，可以生成出一些很美的眼瞳，就像寶石一樣，筆者大推！

# 警語

雖然 AI 生圖很方便，但還是要注意著作權的問題，有些模型訓練是根據某繪師的風格訓練出來的，有可能造成侵權問題，使用前請留意模型下的資訊。

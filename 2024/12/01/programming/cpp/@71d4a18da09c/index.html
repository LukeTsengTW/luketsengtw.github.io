# жҢҮжЁҷпјҲPointerпјү

жҲ‘еҖ‘иҰҒеӯҳеҸ–и®Ҡж•ёзҡ„иЁҳжҶ¶й«”дҪҚеқҖпјҢйңҖиҰҒдҪҝз”ЁгҖҢ&гҖҚз¬ҰиҷҹеҠ еңЁи®Ҡж•ёеҗҚеүҚйқўзҚІеҸ–пјҢеҰӮдёӢпјҡ

NULL жҢҮжЁҷжҳҜе®ҡзҫ©еңЁжЁҷжә–еҮҪејҸеә«дёӯзҡ„еҖјзӮәйӣ¶зҡ„еёёж•ёгҖӮ

# C++ NULL жҢҮжЁҷжҮүз”Ё

  1. еҲқе§ӢеҢ–пјҲInitializationпјүпјҡе°ҮжҢҮжЁҷеҲқе§ӢеҢ–зӮәз©әеҖјпјҲnullptrпјүжҳҜдёҖеҖӢиүҜеҘҪзҡ„зҝ’ж…ЈпјҢеӣ зӮәйҖҷеҸҜд»ҘйҒҝе…ҚжңӘе®ҡзҫ©иЎҢзӮәпјҢдёҰжҳҺзўәиЎЁзӨәжҢҮжЁҷе°ҡжңӘжҢҮеҗ‘жңүж•Ҳзҡ„иЁҳжҶ¶й«”дҪҚзҪ®гҖӮ
  2. й җиЁӯеҖјпјҲDefault Valuesпјүпјҡз•¶жҢҮжЁҷе°ҡжңӘеҲҶй…Қжңүж•Ҳзҡ„иЁҳжҶ¶й«”ең°еқҖжҷӮпјҢз©әжҢҮжЁҷеҸҜдҪңзӮәжҢҮжЁҷзҡ„й җиЁӯеҖјжҲ–еҲқе§ӢеҖјгҖӮ
  3. йҢҜиӘӨиҷ•зҗҶпјҲError Handlingпјүпјҡз©әжҢҮжЁҷеңЁйҢҜиӘӨжўқд»¶дёӢжҲ–з”Ёж–јиЎЁзӨәиіҮж–ҷзјәеӨұжҷӮйқһеёёжңүз”ЁпјҢжңүеҠ©ж–јжӣҙеҘҪең°иҷ•зҗҶдҫӢеӨ–жғ…жіҒгҖӮ
  4. иіҮжәҗйҮӢж”ҫпјҲResource ReleaseпјүпјҡеңЁйҮӢж”ҫиіҮжәҗпјҲдҫӢеҰӮйЎһеҲҘзҡ„и§Јж§ӢеҮҪж•ёпјүжҲ–еҲӘйҷӨеҫҢе°ҮжҢҮжЁҷиЁӯзӮәз©әеҖјпјҲnullptrпјүпјҢеҸҜйҒҝе…Қж„ҸеӨ–дҪҝз”ЁжҲ–еӯҳеҸ–е·ІйҮӢж”ҫзҡ„иЁҳжҶ¶й«”гҖӮ
  5. е“Ёе…өеҖјпјҲSentinel Valuesпјүпјҡз©әжҢҮжЁҷеҸҜд»Ҙз”ЁдҫҶиЎЁзӨәиіҮж–ҷзөҗж§ӢжҲ–жё…е–®зҡ„зөҗе°ҫгҖӮдҫӢеҰӮпјҢеңЁйҸҲзөҗдёІеҲ—дёӯпјҢжңҖеҫҢдёҖеҖӢзҜҖй»һзҡ„гҖҢдёӢдёҖеҖӢзҜҖй»һгҖҚж¬„дҪҚйҖҡеёёжҳҜдёҖеҖӢз©әжҢҮжЁҷгҖӮ

# C++ NULL жҢҮжЁҷзјәй»һ

  • жҸҗеҸ–пјҲdereferenceпјүз©әжҢҮжЁҷжңғе°ҺиҮҙжңӘе®ҡзҫ©иЎҢзӮәпјҲundefined behaviorпјүпјҢеҸҜиғҪеј•зҷјеҹ·иЎҢжҷӮйҢҜиӘӨпјҢдҫӢеҰӮиЁҳжҶ¶й«”еҚҖж®өйҢҜиӘӨпјҲsegmentation faultпјүгҖӮ
  • еңЁжҸҗеҸ–з©әжҢҮжЁҷд№ӢеүҚпјҢйңҖиҰҒжҳҺзўәжӘўжҹҘжҳҜеҗҰзӮәз©әжҢҮжЁҷпјҢжүҚиғҪйҒҝе…ҚжңӘе®ҡзҫ©иЎҢзӮәгҖӮ

иҮіж–јйҖҷй»һпјҢеҸҜд»ҘйҖҸйҒҺжўқд»¶еҲӨж–·зҡ„ж–№ејҸйҒҝе…ҚжӯӨйЎһеһӢйҢҜиӘӨпјҡ

еңЁеӨ§еӨҡж•ёзҡ„дҪңжҘӯзі»зөұдёҠпјҢзЁӢејҸдёҚе…ҒиЁұеӯҳеҸ–дҪҚеқҖзӮә 0 зҡ„иЁҳжҶ¶й«”пјҢеӣ зӮәи©ІиЁҳжҶ¶й«”жҳҜдҪңжҘӯзі»зөұдҝқз•ҷзҡ„гҖӮз„¶иҖҢпјҢиЁҳжҶ¶й«”дҪҚеқҖ 0 жңүзү№еҲҘйҮҚиҰҒзҡ„ж„Ҹзҫ©пјҢе®ғиЎЁжҳҺи©ІжҢҮжЁҷдёҚжҢҮеҗ‘дёҖеҖӢеҸҜеӯҳеҸ–зҡ„иЁҳжҶ¶й«”дҪҚеқҖгҖӮдҪҶжҢүз…§ж…ЈдҫӢпјҢеҰӮжһңжҢҮжЁҷеҢ…еҗ«з©әеҖјпјҲйӣ¶еҖјпјүпјҢеүҮеҒҮе®ҡе®ғдёҚжҢҮеҗ‘д»»дҪ•жқұиҘҝгҖӮ

еӣ жӯӨпјҢеҰӮжһңжүҖжңүжңӘдҪҝз”Ёзҡ„жҢҮжЁҷйғҪиў«иіҰдәҲз©әеҖјпјҢеҗҢжҷӮйҒҝе…ҚдҪҝз”Ёз©әжҢҮжЁҷпјҢе°ұеҸҜд»ҘйҳІжӯўиӘӨз”ЁдёҖеҖӢжңӘеҲқе§ӢеҢ–зҡ„жҢҮжЁҷгҖӮеҫҲеӨҡжҷӮеҖҷпјҢжңӘеҲқе§ӢеҢ–зҡ„и®Ҡж•ёеӯҳжңүдёҖдәӣеһғеңҫеҖјпјҢе°ҺиҮҙзЁӢејҸйӣЈд»ҘиӘҝи©ҰгҖӮ

# жҢҮжЁҷзҡ„з®—иЎ“йҒӢз®—пјҲPointer Arithmeticпјү

жҢҮжЁҷжҳҜдёҖеҖӢз”Ёж•ёеҖјиЎЁзӨәзҡ„дҪҚеқҖгҖӮеӣ жӯӨпјҢжӮЁеҸҜд»Ҙе°ҚжҢҮжЁҷеҹ·иЎҢз®—иЎ“йҒӢз®—гҖӮжҢҮжЁҷеҸҜд»ҘйҖІиЎҢеӣӣзЁ®з®—иЎ“йҒӢз®—пјҡ++гҖҒ--гҖҒ+гҖҒ-гҖӮ

# зёҪзөҗ

жң¬ж–Үд»Ӣзҙ№дәҶ C++ дёӯзҡ„жҢҮжЁҷпјҲPointerпјүпјҢеҢ…жӢ¬жҢҮжЁҷзҡ„еҹәжң¬жҰӮеҝөгҖҒдҪҝз”Ёж–№жі•еҸҠзӣёй—ңйҒӢз®—пјҢдёҰжҺўиЁҺз©әжҢҮжЁҷпјҲNull PointerпјүеҸҠе…¶жҮүз”ЁгҖӮ

# жҢҮжЁҷзҡ„еҹәжң¬жҰӮеҝө

  • жҢҮжЁҷпјҲPointerпјүпјҡжҢҮжЁҷжҳҜдёҖзЁ®и®Ҡж•ёпјҢз”Ёж–је„Іеӯҳе…¶д»–и®Ҡж•ёзҡ„иЁҳжҶ¶й«”дҪҚеқҖгҖӮе…¶иӘһжі•зӮә type *ptr; гҖӮ
  • иЁҳжҶ¶й«”дҪҚеқҖзҚІеҸ–пјҡдҪҝз”Ё & з¬ҰиҷҹдҫҶзҚІеҸ–и®Ҡж•ёзҡ„иЁҳжҶ¶й«”дҪҚеқҖгҖӮ
  • жҸҗеҸ–йҒӢз®—еӯҗпјҲDereferenceпјүпјҡ* з”Ёж–јжҸҗеҸ–жҢҮжЁҷжүҖжҢҮеҗ‘зҡ„еҖјгҖӮ

# з©әжҢҮжЁҷпјҲNull Pointerпјү

  • е®ҡзҫ©пјҡе…¶еҖјзӮә NULLпјҢиЎЁзӨәдёҚжҢҮеҗ‘д»»дҪ•жңүж•Ҳзҡ„иЁҳжҶ¶й«”дҪҚеқҖгҖӮ

# гҖҗC++ зӯҶиЁҳгҖ‘йҷЈеҲ—пјҲArrayпјү - part 8

дҪ еҘҪпјҢжҲ‘дёҰдёҚжҳҜд»Җйәј C++гҖҒзЁӢејҸиӘһиЁҖзҡ„е°Ҳе®¶пјҢжүҖд»Ҙжң¬ж–ҮиӢҘжңүдәӣйҢҜиӘӨйә»з…©и«Ӣеҗ„дҪҚйһӯеӨ§еҠӣдёҖй»һпјҢжҲ‘жҘөйңҖеҗ„дҪҚзҡ„жҢҮжӯЈеҸҠжҢҮе°ҺпјҒпјҒжң¬зі»еҲ—ж–Үз« зҡ„жҖ§иіӘдё»иҰҒд»Ҙи©ји«§зҡ„еҸЈеҗ»пјҢдёҖжҙҫиј•й¬Ҷзҡ„ж…ӢеәҰиҮӘеӯёзЁӢејҸиӘһиЁҖпјҢеҰӮжһңдҪ е–ңжӯЎпјҢйә»з…©з•ҷиЁҖиӘӘиҒІж–Үз« и®ҡи®ҡеҗ§пјҒ

# йҷЈеҲ—пјҲArrayпјү

йҷЈеҲ—пјҲArrayпјүпјҢеҸҲзЁұж•ёзө„пјҢеңЁ Python дёӯпјҢжӯӨзЁ®йЎһдјјзҡ„иіҮж–ҷзөҗж§ӢзЁұзӮәеҲ—иЎЁпјҲListпјүпјҢжҳҜе°ҮдёҖзө„иіҮж–ҷе„ІеӯҳеңЁиЁҳжҶ¶й«”зҡ„йӣҶеҗҲгҖӮ

йҷЈеҲ—жҳҜдёҖзЁ®йӣҶеҗҲпјҢеңЁ C++ дёӯеҸӘиғҪе„Іеӯҳеӣәе®ҡиіҮж–ҷеһӢж…Ӣзҡ„е…ғзҙ ж–јйҷЈеҲ—дёӯгҖӮ

еңЁжӯӨд№ӢеүҚпјҢжҲ‘еҖ‘йңҖиҰҒдәҶи§ЈйҷЈеҲ—пјҲArrayпјүгҖҒе…ғзҙ пјҲElementпјүгҖҒзҙўеј•пјҲIndexпјүгҖҒиіҮж–ҷеһӢж…ӢпјҲData Typeпјүд№Ӣй–“зҡ„й—ңдҝӮгҖӮ

Image Sourceпјҡhttps://www.geeksforgeeks.org/cpp-arrays/

д»ҘдёҠжҳҜйҷЈеҲ—зҡ„ең–зӨәпјҢжүҖи¬Ӯзҡ„е…ғзҙ е°ұеғҸжҳҜж•ёеҲ—иЈЎйқўзҡ„еҗ„еҖӢж•ёеӯ—дёҖжЁЈпјҢеҰӮпјҡ{1,2,3,4,5,6}пјҢ1~6 еҚізӮәе…ғзҙ гҖӮ

жүҖи¬Ӯзҡ„зҙўеј•пјҲIndexпјүпјҢе°ұжҳҜжҜҸеҖӢе…ғзҙ зҡ„дҪҚзҪ®пјҢC++ зҡ„йҷЈеҲ—зҙўеј•д»Ҙ 0 зӮәй–Ӣй ӯпјҢдёҚжҳҜ 1 е”·гҖӮ

д»ҘдёҠең–иҖҢиЁҖпјҢе…ғзҙ  2 зҡ„зҙўеј•е°ұжҳҜ 0пјҢе®ғжүҖеңЁзҡ„дҪҚзҪ®е°ұзЁұзӮә 0гҖӮ

иҮіж–јиіҮж–ҷеһӢж…ӢпјҢйҷЈеҲ—дёҚйңҖиҰҒзӮәжҜҸеҖӢе…ғзҙ еҠ дёҠ intгҖҒfloat зӯүзӯүпјҢеҸӘйңҖиҰҒеңЁйҷЈеҲ—еҗҚзЁұеүҚеҠ дёҠеҚіеҸҜпјҢиӘһжі•еҰӮдёӢпјҡ

typeпјҡиіҮж–ҷеһӢж…Ӣ
arraysizeпјҡжҢҮе®ҡйҷЈеҲ—еӨ§е°ҸпјҢеҝ…й ҲжҳҜиҰҒ > 0 зҡ„ж•ҙж•ёеёёж•ёгҖӮ

е…·й«”иҖҢиЁҖпјҢжӯЈеёёзҡ„е®Је‘ҠеҰӮдёӢпјҡ

йҖҷжҳҜж•ҙж•ёеһӢж…Ӣзҡ„йҷЈеҲ— aпјҢеӨ§е°ҸиғҪеӨ еЎһ 10 еҖӢе…ғзҙ йҖІеҺ»пјҢзҙўеј•зӮә 0 ~ 9гҖӮ

# еҲқе§ӢеҢ–йҷЈеҲ—

д»ҘдёҠе®ҡзҫ©дәҶдёҖеҖӢйҷЈеҲ—пјҢдёҰдё”еҒҡеҮәеҲқе§ӢеҢ–зҡ„еӢ•дҪңгҖӮ

йӮЈжҲ‘еҖ‘д№ҹеҸҜд»ҘдёҚз”ЁжҢҮе®ҡ arraysizeпјҢеҰӮдёӢпјҡ

еүҮ C++ жңғиҮӘеӢ•еҒөжё¬иЈЎйқўзҡ„е…ғзҙ еҖӢж•ёпјҢдёҰдё”еҲ¶е®ҡеӨ§е°ҸзӮә 10пјҢеҗҢдёҠејҸгҖӮ

жҲ‘еҖ‘д№ҹеҸҜд»ҘзӮәйҷЈеҲ—жҢҮе®ҡеҖјпјҢеҰӮдёӢпјҡ

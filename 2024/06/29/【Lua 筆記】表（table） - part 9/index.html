# иЎЁпјҲTableпјү

Lua дёӯзҡ„иЎЁпјҲtableпјүжҳҜдёҖзЁ®еҫҲеј·еҫҲеҘҪз”Ёзҡ„иіҮж–ҷзөҗж§ӢпјҢдё»иҰҒжңүд»ҘдёӢе№ҫеҖӢзү№й»һпјҡ

  • Dynamicпјҡtable еҸҜд»ҘеӢ•ж…Ӣең°ж–°еўһжҲ–еҲӘйҷӨй …зӣ®пјҲitemпјүпјҢз„ЎйңҖдәӢе…Ҳе®Је‘Ҡ table зҡ„еӨ§е°ҸгҖӮйҖҷи®“ table йқһеёёйқҲжҙ»пјҢеҸҜд»Ҙз”Ёж–јеҗ„зЁ®иіҮж–ҷзөҗж§ӢпјҢеҰӮйҷЈеҲ—гҖҒеӯ—е…ёзӯүгҖӮ
  • Key / ValueпјҲйҚөеҖје°Қпјүпјҡtable дёӯзҡ„иіҮж–ҷжҳҜд»ҘйҚөеҖје°Қзҡ„еҪўејҸйҖІиЎҢе„Іеӯҳзҡ„гҖӮйҚөпјҲkeyпјүеҸҜд»ҘжҳҜд»»ж„ҸеһӢж…Ӣзҡ„еҖјпјҲйҷӨдәҶ nilпјүпјҢеҢ…жӢ¬ж•ёеӯ—гҖҒеӯ—дёІжҲ–иҖ… table жң¬иә«зӯүгҖӮеҖјпјҲvalueпјүд№ҹеҸҜд»ҘжҳҜд»»ж„ҸеһӢж…Ӣзҡ„гҖӮ
  • DifferenceпјҡиЎЁдёӯеҸҜд»ҘеҗҢжҷӮе„ІеӯҳдёҚеҗҢеһӢж…Ӣзҡ„еҖјпјҢиЎЁзӨәеңЁеҗҢдёҖеҖӢ table дёӯеҸҜд»ҘеҗҢжҷӮжңүж•ҙж•ёгҖҒеӯ—дёІгҖҒеҮҪж•ёзӯүдёҚеҗҢеһӢж…Ӣзҡ„иіҮж–ҷгҖӮ
  • IndexпјҡLua дёӯзҡ„ table зҙўеј•пјҲindexпјүеҸҜд»ҘжҳҜж•ҙж•ёжҲ–иҖ…еӯ—дёІпјҢйҖҷи®“ table ж—ўеҸҜд»ҘдҪңзӮәйҷЈеҲ—дҪҝз”ЁпјҢд№ҹеҸҜд»ҘдҪңзӮәеӯ—е…ёпјҲжҲ–зЁұе“ҲеёҢиЎЁгҖҒжҳ е°„пјүдҪҝз”ЁгҖӮз•¶дҪҝз”Ёж•ҙж•ёдҪңзӮәзҙўеј•жҷӮпјҢиЎЁзҸҫеҪўејҸе’ҢеҠҹиғҪйЎһдјјж–јеӮізөұзҡ„йҷЈеҲ—гҖӮ
  • Automatically Adjusting Sizeпјҡз•¶иіҮж–ҷиў«ж–°еўһеҲ° table з•¶дёӯжҷӮпјҢLua жңғиҮӘеӢ•иӘҝж•ҙ table зҡ„еӨ§е°ҸгҖӮ

зёҪд№Ӣ table иҰҒиӘӘеғҸжҳҜ python з•¶дёӯзҡ„еӯ—е…ёпјҢеҘҪеғҸд№ҹдёҚе®Ңе…ЁжҳҜпјҢеӣ зӮәд»–еҸҜд»Ҙи®ҠдҫҶи®ҠеҺ»зҡ„пјҢйқһеёёйқҲжҙ»гҖӮ

# иЎЁпјҲtableпјүMethod

иЎЁж јдҫҶжәҗпјҡLua table (иЎЁ) | иҸңйёҹж•ҷзЁӢ

Numberж–№жі• & з”ЁйҖ”
1table.concat (table [, sep [, start [, end]]])пјҡconcat жҳҜ concatenateпјҲйҖЈйҺ–гҖҒйҖЈжҺҘпјүзҡ„зё®еҜ«гҖӮtable.concat () еҮҪж•ёеҲ—еҮәеҸғж•ёдёӯжҢҮе®ҡ table зҡ„йҷЈеҲ—йғЁеҲҶеҫһ start дҪҚзҪ®еҲ° end дҪҚзҪ®зҡ„жүҖжңүе…ғзҙ пјҢе…ғзҙ й–“д»ҘжҢҮе®ҡзҡ„еҲҶйҡ”з¬ҰиҷҹпјҲsepпјүйҡ”й–ӢгҖӮ
2table.insert (table, [pos,] value)пјҡеңЁ table зҡ„йҷЈеҲ—йғЁеҲҶжҢҮе®ҡдҪҚзҪ®пјҲposпјүжҸ’е…ҘеҖјзӮә value зҡ„дёҖеҖӢе…ғзҙ гҖӮpos еҸғж•ёеҸҜйҒёпјҲoptionalпјүпјҢй җиЁӯзӮәйҷЈеҲ—йғЁеҲҶзөҗжқҹгҖӮ
3table.maxn (table)пјҡжҢҮе®ҡ table дёӯжүҖжңүжӯЈж•ё key еҖјдёӯжңҖеӨ§зҡ„ key еҖјгҖӮеҰӮжһңдёҚеӯҳеңЁ key еҖјзӮәжӯЈж•ёзҡ„е…ғзҙ пјҢеүҮеӣһеӮі 0гҖӮпјҲLua5.2 д№ӢеҫҢжӯӨж–№жі•е·Із¶“дёҚеӯҳеңЁдәҶпјҢжң¬ж–ҮдҪҝз”ЁдәҶиҮӘиЁӮеҮҪж•ёеҜҰдҪңпјү
4table.remove (table [, pos])пјҡеӣһеӮі table йҷЈеҲ—йғЁеҲҶдҪҚж–ј pos дҪҚзҪ®зҡ„е…ғзҙ гҖӮе…¶еҫҢзҡ„е…ғзҙ жңғиў«еүҚз§»гҖӮpos еҸғж•ёеҸҜйҒёпјҢй җиЁӯзӮә table й•·еәҰпјҢеҚіеҫһжңҖеҫҢдёҖеҖӢе…ғзҙ еҲӘиө·гҖӮ
5table.sort (table [, comp])пјҡе°ҚзөҰе®ҡзҡ„ table йҖІиЎҢеҚҮеәҸжҺ’еәҸгҖӮ

table.sort (table [, comp])пјҢ[, comp] зӮәжҜ”ијғеҮҪж•ёзҡ„ең°ж–№пјҢеңЁйҖҷйӮҠдҪҝз”ЁдёҖеҖӢеҢҝеҗҚеҮҪж•ё function (a, b) return a > b endпјҢйҖҷеҖӢеҮҪж•ёжңғеңЁжҺ’еәҸйҒҺзЁӢдёӯжҜ”ијғе…©еҖӢе…ғзҙ пјҢдё”еңЁз¬¬дёҖеҖӢе…ғзҙ еӨ§ж–јз¬¬дәҢеҖӢе…ғзҙ жҷӮеӣһеӮі trueпјҢеҫһиҖҢеҜҰзҸҫйҷҚеәҸпјҲз”ұеӨ§еҲ°е°ҸпјүжҺ’еәҸгҖӮ

еҢҝеҗҚеҮҪж•ёпјҲAnonymous functionsпјүпјҡжІ’жңүеҗҚзЁұзҡ„еҮҪж•ёпјҢйҖҡеёёжһ¶ж§ӢијғзӮәз°Ўз•ҘпјҢеӨ§еӨҡжғ…жіҒиғҪеӨ жҸҗеҚҮзЁӢејҸзўјж•ҲзҺҮгҖӮиЈңе……пјҡеңЁ Python еүҮжҳҜз”Ё lambda дҪңзӮәеҢҝеҗҚеҮҪж•ёзҡ„й—ңйҚөеӯ—гҖӮ

# зёҪзөҗ

  • DynamicпјҡеҸҜд»ҘеӢ•ж…Ӣж–°еўһжҲ–еҲӘйҷӨй …зӣ®пјҢз„ЎйңҖдәӢе…Ҳе®Је‘ҠеӨ§е°ҸгҖӮ
  • Key / ValueпјҡиіҮж–ҷд»ҘйҚөеҖје°ҚеҪўејҸе„ІеӯҳпјҢйҚөе’ҢеҖјеҸҜд»ҘжҳҜд»»ж„ҸеһӢж…ӢпјҲйҷӨдәҶ nilпјүгҖӮ
  • DifferenceпјҡеҗҢдёҖеҖӢ table дёӯеҸҜд»Ҙе„ІеӯҳдёҚеҗҢеһӢж…Ӣзҡ„еҖјгҖӮ
  • Indexпјҡзҙўеј•еҸҜд»ҘжҳҜж•ҙж•ёжҲ–еӯ—дёІпјҢж—ўеҸҜдҪңзӮәйҷЈеҲ—д№ҹеҸҜдҪңзӮәеӯ—е…ёдҪҝз”ЁгҖӮ
  • Automatically Adjusting Sizeпјҡж–°еўһиіҮж–ҷжҷӮпјҢLua жңғиҮӘеӢ•иӘҝж•ҙ table зҡ„еӨ§е°ҸгҖӮ
